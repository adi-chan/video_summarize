VIDEO SUMMARIZER – DOCUMENTATION

1. OVERVIEW

This project is a command-line video summarizer that accepts a YouTube link or a local media file (.mp4 / .mp3), optionally trims a specific time segment, converts audio to WAV, transcribes speech to text, and generates a summary.

The tool is designed to work offline-first (using Whisper) and can later be extended to online APIs.

2. COMMAND LINE USAGE

Using Docker(recommended):

docker run -it \
          -v ~/.cache/huggingface:/root/.cache/huggingface \
          -v $(pwd)/data:/app/data \
          video_summarize \
          python main.py 'YOUTUBE LINK' [--start mm:ss] [--end mm:ss]

Using python env:

- python main.py <input> [--start mm:ss] [--end mm:ss]

Examples:

Summarize full YouTube video:

Docker:
docker run -it \
          -v ~/.cache/huggingface:/root/.cache/huggingface \
          -v $(pwd)/data:/app/data \
          video_summarize \
          python main.py 'https://www.youtube.com/watch?v=6Jk0zgNmXZ4' 

Python env:
python main.py https://youtube.com/watch?v=xxxx

Summarize a specific segment:

Docker:
docker run -it \
          -v ~/.cache/huggingface:/root/.cache/huggingface \
          -v $(pwd)/data:/app/data \
          video_summarize \
          python main.py 'https://www.youtube.com/watch?v=6Jk0zgNmXZ4' --end 8:48

Python env:

python main.py video.mp4 --start 0:10 --end 0:30

Start time only (till end):

Docker:
docker run -it \
          -v ~/.cache/huggingface:/root/.cache/huggingface \
          -v $(pwd)/data:/app/data \
          video_summarize \
          python main.py 'https://www.youtube.com/watch?v=6Jk0zgNmXZ4' --start 2:34

Python env:
python main.py audio.mp3 --start 45

3. ARGUMENT DETAILS

The optional arguments (--start and --end) are not mandatory.
If the user wants to summarize the entire video, no additional arguments are required.

Simply Use: 

python main.py "URL/path_to_file"

4. PERFORMANCE

Measured on CPU (offline mode):

~3.5× real-time processing speed
(Processes video in ~28–30% of its duration)

~80–85% semantic alignment with reference summaries (offline Transformer model)

Online (OpenAI API):

~90%+ semantic alignment

Higher structural clarity and abstraction quality

Performance depends on:

Video length

Model size

Hardware specifications

The summarization model can be modified inside summarize.py.

Users may configure CPU or GPU execution depending on system capabilities.
